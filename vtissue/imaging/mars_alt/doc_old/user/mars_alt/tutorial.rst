Welcome to the MARS_ALT tutorial
################################

In this tutorial, you will learn how to fuse images together, how to segment them
and finally how to track cell lineages between acquisitions.

You can install the data we will be using like this:

.. code-block:: bash

    $> alea_install VPlants.VTissueData
