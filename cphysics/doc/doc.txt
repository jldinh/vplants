Technical documentation
-----------------------

C++ documentation
-----------------
The documentation for the C++ library has been generated with doxygen : 

http://www.stack.nl/~dimitri/doxygen/

To regenerate it, install Doxygen on your system and use the provide Doxyfile.
You can also use doxywizard which is a graphical front end for doxygen.

Python documentation
--------------------
The python documentation has been generated with epydoc.

http://epydoc.sourceforge.net/

You can use epydoc_gui which is a graphical front end for epydoc.
