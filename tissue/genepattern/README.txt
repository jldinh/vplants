===== About =====

=== Description ===

genepattern is a python package that describe some abstract geometrical objects and ways to use them.

 
=== Content ===

The genepattern package contains :

  *Meta informations files (README.TXT, AUTHORS.TXT, LICENSE.TXT).
  *Python package (src/genepattern).
	- interface definition for abstract geometrical objects
	- a set of algo to manipulate these objects

===== Installation =====

=== Download ===

use svn and check out the genepattern directory

=== Requirements ===

* Python 2.4 or higher

=== Installation ===

python setup.py install
