Drawing of a tissue
This package is an example on how to draw a 2D tissue using svg format
Direct dependencies :
  • celltissue
  • svgdraw
  • pglviewer
  • plantgl

Description

This package provide an example on how to use svgdraw to construct a tissue from a 2D vectorial drawing.

Algorithms

Simulation

This package provide a simple algorithm to translate a vectorial drawing into a tissue. First use "create_tissue.py" to create a tissue from "coleochaete.svg" file.
user@computer:$ python create_tissue.py
You can then run "simu.py" to display the created tissue.
user@computer:$ python simu.py
The view is a classical PyQGLViewer 3D view stuck in 2D so you cannot rotate it. The GUI propose in the toolbar a set of icons that allow to display each kind of element of the tissue independently: points, walls and cells. The last one display or not the background image.

Exercices

To go further :
  • open the svg file (use inkscape for example) and try to modify it. Then rerun the script to appreciate the modifications
  • Try to add some information in the svg file and retrieve it as a property in the tissue
